watchIt-client (Angular)

An alternative implementation of the watchIt client allowing the users to create a profile, log in, see a list of movies and the individual information of the movies, as well as create a custom watchlist among other functions. It is built using Angular and Typescript. The app works with the watchIt backened and is styled using Angular Material.

angular, angular material, typescript (and typedoc)
